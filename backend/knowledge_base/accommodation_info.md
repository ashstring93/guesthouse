# 숙소 기본 정보

## 물레방아하우스 (Mullebang-a House)

### 위치
- **주소**: 전라북도 전주시 완산구
- **접근성**: 
  - 전주 한옥마을: 도보 10-15분 / 차량 5분
  - 객리단길: 도보 10-15분
  - 풍패지관: 근처
  - 전주역/고속터미널: 택시로 10-15분

### 숙소 유형
- 마당이 있는 단독주택
- 프라이빗한 독채 숙소
- 도심 속 조용한 힐링 공간

### 수용 인원
- 최대 인원: 6명
- 침실 2개
  - 메인 침실: 퀸 사이즈 침대 1개
  - 두 번째 침실: 싱글 침대 1개
- 욕실 2개

### 가격
- **1박**: 210,000원
- 바비큐 이용 시 추가 요금 발생

### 주요 시설
- ✨ 침실 2개 (퀸1, 싱글1)
- ✨ 욕실 2개
- ✨ 무료 주차 공간
- ✨ 여행 가방 보관 가능
- ✨ 넓은 잔디 마당
- ✨ 바비큐 시설
- ✨ 현대적인 주방
- ✨ 아늑한 거실
- ✨ 빈티지 인테리어

### 주차
- 숙소 주변 무료 주차 공간 제공
- 차량 진입 가능

### 주변 편의시설
- 바로 앞: 편의점
- 주변: 아파트 단지 (조용한 주택가)
- 근처: 각종 관공서 및 식당

### 특징
- 잘 정돈된 잔디 마당
- 사방이 막힌 프라이빗한 공간
- 대추나무와 감나무가 있는 정원
- 한옥 감성과 현대적 편의 조화
- 청결하고 깨끗한 관리
