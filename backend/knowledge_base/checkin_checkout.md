# 체크인 / 체크아웃 안내

## 체크인

### 시간
- **체크인 시간**: 15:00
- **방식**: 셀프 체크인

### 절차
1. 호스트에게 도착 시간 사전 안내
2. 숙소 위치 안내 받기
3. 비밀번호 또는 열쇠 안내 받기
4. 자율적으로 체크인

### 사전 준비
- 짐 보관: 체크인 시간 전 도착 시 여행 가방 보관 가능
- 사전 문의 필요

### 늦은 체크인
- 가능 여부는 호스트와 사전 협의 필요

3. 사용한 숙소 상태 원복
